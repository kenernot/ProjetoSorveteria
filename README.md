# Projeto integrador

## Descrição

- implementação de um sistema de PDV para sorveterias

## Requisitos

- Cadastro de Usuário/Funcionário
- Cadastro de Produto
- Cadastro de Cliente
- PDV
- Abertura de caixa
- Registro de Caixa
- Fechamento de caixa



## Configurações do banco

- Criar um arquivo db.properties na raiz do projeto dentro da pasta FONTES
- Conteúdo do arquivo deve ser da seguinte forma:
```
user=seu usuário
password=sua senha
connurl=jdbc:mysql://localhost:3306/nome do banco
conndriver=com.mysql.jdbc.Driver
```
